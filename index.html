Sistem pertama akan *Mengekstrak* data dari input (simulasi pemindaian QR). Kemudian, sistem *Mentransformasi* data untuk mengidentifikasi apakah itu kode QRIS atau URL. Data QRIS di *Simpan & Dimuat* sebagai referensi sebelum *Deteksi*. URL langsung menjalani *Analisis* untuk reputasi. Kedua jalur ini mengarah pada hasil *Deteksi* akhir: BERBAHAYA/VALID untuk QRIS, dan BERBAHAYA/AMAN untuk URL.
